About

The Koch Snowflake is a very intriguing math concept that has caught my interest as of recent. Essentially, it is math-curve, a fractal, which was brought to attention by Helge von Koch. The main idea behind it, is the fact that it is possible to have a shape with an infinite perimeter but a finite area. This to me, was a very interesting concept to ponder upon.

There's only so many iterations I can calculate manually, so, out of interest, I decided make something to calculate the iterations for me. Before this web-version, I made a Java version.
